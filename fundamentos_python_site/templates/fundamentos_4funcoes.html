Fundamentos da Programação em Python

Funções e Procedimentos

Conceito

Funções agrupam blocos de código que podem ser reutilizados. Procedimentos são funções que não retornam valor.

Aplicação

Melhoram a organização do código e permitem reutilização de lógica.

Exemplo de código

def saudacao(nome): print(f"Olá, {nome}!") saudacao("Maria")
