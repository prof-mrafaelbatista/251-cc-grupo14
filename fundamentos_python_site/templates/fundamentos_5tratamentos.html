Fundamentos da Programação em Python

Tratamentos de Exceção

Conceito

Permitem lidar com erros de forma controlada, evitando que o programa quebre inesperadamente.

Aplicação

Útil ao lidar com entrada de dados do usuário, operações de arquivos, divisão por zero, etc.

Exemplo de código

try: numero = int(input("Digite um número: ")) resultado = 10 / numero print("Resultado:", resultado) except ValueError: print("Por favor, digite um número válido.") except ZeroDivisionError: print("Não é possível dividir por zero.")
