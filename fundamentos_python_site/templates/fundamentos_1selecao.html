Estruturas de Seleção

Conceito

Permitem que o programa tome decisões com base em condições. As mais comuns são as estruturas if, elif e else.

Aplicação

Usadas quando precisamos executar determinados blocos de código apenas se certas condições forem verdadeiras.

Exemplo de código

if idade >= 18: print("Você é maior de idade.") else: print("Você é menor de idade.")
