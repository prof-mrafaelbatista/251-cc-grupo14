Vetores e Matrizes

Conceito

Vetores são listas unidimensionais e matrizes são listas de listas (bidimensionais).

Aplicação

Utilizados para armazenar e manipular coleções de dados, como notas de alunos, nomes, ou dados em tabelas.

Exemplo de código

vetor = [1, 2, 3] matriz = [[1, 2], [3, 4]] print(vetor[0]) print(matriz[1][1])
