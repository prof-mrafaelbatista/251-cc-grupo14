Estruturas de Repetição

Conceito

Permitem executar um bloco de código várias vezes. As principais são for e while.

Aplicação

São usadas para percorrer listas, repetir ações enquanto uma condição for verdadeira, entre outros.

Exemplo de código

for i in range(5): print("Repetição:", i)
